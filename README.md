# Auto-Install Docker sur Ubuntu

Ce dépôt contient un script Bash pour installer automatiquement Docker (et ses outils associés) sur une machine Ubuntu.

## Prérequis
- Une machine Ubuntu (testé sur Ubuntu 22.04 LTS)
- Accès sudo
